# 🎵 Melodio

> Your cozy corner of the internet. Stream lofi beats perfect for studying, working, or relaxing.

A warm, aesthetic lofi music player built with Zo Computer for the Zo Hackathon 2026.

![Melodio Screenshot](https://raw.githubusercontent.com/4yyb/melodio/main/public/images/melodio-coffee-shop.png)

## ✨ Features

- 🌅 **Animated Sunset Background** - Warm gradient with floating orbs and particles
- 🎵 **4 Lofi Channels** - Study, Chill, Sleep, and Work stations
- 🎧 **SoundCloud Integration** - High-quality lofi streams
- 🎙️ **Live Visualizer** - Animated audio visual bars
- 💿 **Vinyl Spinning Animation** - Retro aesthetic touch
- 📱 **Fully Responsive** - Works on desktop and mobile
- 🔊 **Volume Control** - Custom styled slider

## 🎨 Design

- **Aesthetic**: Cozy coffee shop vibes, anime chibi characters
- **Color Palette**: Warm sunset gradient (orange → pink → purple)
- **Typography**: Outfit (Google Fonts)

## 🚀 Tech Stack

- **Framework**: React + Vite + TypeScript
- **Styling**: Tailwind CSS 4 + custom animations
- **Hosting**: Zo Computer (Site)
- **Audio**: SoundCloud Embed

## 📦 Deployment

```bash
bun run build
bun run prod
```

Site is live at: **https://ayb.zocomputer.io**
